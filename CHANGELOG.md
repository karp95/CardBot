# Changelog

Все значимые изменения в проекте документируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/).

## [0.2.0] - 2026-02-24

### Добавлено
- Стандартные наборы по топикам из БД (template_sets, template_cards)
- 8 топиков по 50 слов: Животные, Еда и напитки, Семья, Дом, Цвета, Путешествия, Работа, Природа
- Кнопка «Добавить стандартные наборы» в разделе «Наборы»
- Массовая загрузка карточек через /add (несколько строк за раз)
- 4 режима обучения: EN→RU, RU→EN, ввод RU→EN, ввод EN→RU
- Меню команд (SetMyCommands) при вводе /

### Изменено
- DefaultTopicsService читает шаблоны из БД вместо хардкода

### Удалено
- Устаревшие скрипты import_top100_words.sql, import_top100_words_user1.sql

## [0.1.0] - 2026-02-24

### Добавлено
- Команды: /start, /add, /learn, /list, /sets, /stats, /help, /cancel
- Добавление карточек (слово — перевод)
- Режим обучения с inline-кнопками
- Наборы карточек
- Редактирование и удаление карточек
- Ежедневные напоминания
- Docker Compose для развёртывания
